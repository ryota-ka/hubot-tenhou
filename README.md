# hubot-tenhou

Draws 14 mahjong tiles

See [`src/tenhou.js`](src/tenhou.js) for full documentation.

## Installation

In hubot project repo, run:

`npm install hubot-tenhou --save`

Then add **hubot-tenhou** to your `external-scripts.json`:

```json
["hubot-tenhou"]
```

## Sample Interaction

```
you>> hubot tenhou
hubot>>🀁🀁🀅🀉🀏🀑🀖🀘🀙🀛🀝🀞🀟🀠
```
